**Goat服务器引擎**

####  主要特性

- 跨平台的服务器框架

- 网络层使用高效的libuv库

- 事件驱动模型,单进程异步方式

- 支持异步mysql访问

- 支持异步redis访问

- 支持内嵌lua，逻辑层使用lua开发

- 支持多协议(tcp+二进制 ,tcp+json websocket+二进制，websocket+json)

- 支持部署和调试两种模式

   部署模式：服务集群之间通过netbus模块转发消息

   开发调试模式：整个集群的逻辑都可以放入一个进程，接入层开发调试

   能极大提高开发调试效率

- 支持lua配置文件
