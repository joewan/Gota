**Goat服务器引擎**

####  主要特性

- 跨平台的服务器框架

- 网络层使用高效的libuv库

- 事件驱动模型,单进程异步方式

- 支持异步mysql

- 支持异步redis

- 支持内嵌lua，导出底层接口，上层业务使用lua开发

- 网络层支持TCP UDP WebSocket协议

- 应用层多协议支持(protobuf ,json)

- 支持部署和调试两种模式

   部署模式：服务集群之间通过netbus模块转发消息

   开发调试模式：整个集群的逻辑都可以放入一个进程，如gatawayserver开发调试

   能极大提高开发调试效率

- 支持lua配置文件
